The following type definitions are available globally.

  • RLMObjectMigrationBlock

    Provides both the old and new versions of an object in this Realm. Objects properties can only be accessed using keyed subscripting.

    Declaration

    Objective-C

    typedef void (^RLMObjectMigrationBlock)(RLMObject *_Nullable,
                                        RLMObject *_Nullable)

    Parameters
